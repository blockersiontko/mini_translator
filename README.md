# Polish to English Animal Dictionary 🐾

A simple Python program that translates Polish animal names into English using a dictionary.

## Features

* Translates animal names from Polish to English.
* Validates user input.
* Repeats the prompt until a valid animal name is entered.
* Uses a Python dictionary for fast lookups.

## Animals Included

| Polish       | English |
| ----------   | ------- |
| kot          | cat     |
| pies         | dog     |
| panda        | panda   |
| lis          | fox     |
| wilk         | wolf    |
| wąż          | snake   |
| niedźwiedź   | bear    |
| koń          | horse   |
| konik morski | seahorse|
| ryba         | fish    |

## Installation

Clone this repository:

```bash
git clone https://github.com/blockersiontko/mini_translator.git
```

Move into the project directory:

```bash
cd mini_translator
```

## How to Run

Run the Python script:

```bash
python mini_translator.py
```

## Example

```text
Wpisz nazwę zwierzęcia po polsku: kot
Zwierzę 'kot' nazywa się po angielsku 'cat'.
```

If an invalid word is entered:

```text
Wpisz nazwę zwierzęcia po polsku: meow
Nie ma tego słowa w moim słowniku!
Wpisz nazwę zwierzęcia po polsku: lis
Zwierzę 'lis' nazywa się po angielsku 'fox'.
```
